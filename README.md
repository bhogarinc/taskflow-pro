# TaskFlow Pro - Intelligent Project Management Platform

## Overview
TaskFlow Pro is an AI-powered project management platform built with Python FastAPI, React, PostgreSQL, and Redis.

## Tech Stack
- **Backend**: Python 3.11, FastAPI, SQLAlchemy, Alembic
- **Frontend**: React 18, TypeScript, TailwindCSS
- **Database**: PostgreSQL 16, Redis 7
- **Infrastructure**: Docker, GitHub Actions, Azure App Service

## Quick Start
```bash
git clone https://github.com/bhogarinc/taskflow-pro.git
cd taskflow-pro
docker compose up -d
```
